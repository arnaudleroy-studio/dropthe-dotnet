# DropThe .NET Client

.NET client library for accessing DropThe open data.

## Overview

DropThe is a data utility media network covering movies, series, crypto, companies, and more.

- **Homepage**: [https://dropthe.org](https://dropthe.org)
- **Documentation**: [https://dropthe.org/docs](https://dropthe.org/docs)
- **API**: [https://dropthe.org/api](https://dropthe.org/api)

## Installation

```bash
dotnet add package DropThe
```

## Quick Start

```csharp
using DropThe;

Console.WriteLine($"DropThe v{Info.Version}");
Console.WriteLine($"API Base: {Info.BaseUrl}");
```

## Data Coverage

- 66,000+ movies and series
- 1,800,000+ entities in the knowledge graph
- Cryptocurrency tracking
- Company profiles
- Streaming availability data
